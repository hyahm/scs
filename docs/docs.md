# 启动本地的scs文档


> 先克隆一份代码到本地
```shell
git clone https://gitee.com/cander/scs.git
// 或 github
git clone https://github.com/hyahm/scs.git
```

> 安装node

http://nodejs.cn/download/

![node下载](/images/node.png)

> 安装一下 docsify
```
npm i docsify-cli -g
```

> 在scs的项目根目录 启动本地文档
```
docsify.cmd serve docs
```
