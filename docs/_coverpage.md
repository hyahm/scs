# scs <small>3.0</small>

> 一个服务管理加服务监控的进程管理工具。

-  为了保护数据， 可以让服务在某段时间内才能停止
-  监控硬件信息， 主要是磁盘， cpu， 内存,  scs服务  
-  服务之间可以相互控制增删改查  
-  报警功能api  
-  支持定时器功能执行命令或脚本
-  客户端控制多台服务器
